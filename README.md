# BallsGame

Учебный проект по дисциплине ООП.

## Задача

Определить количество шариков, которые будут уничтожены после удаления цепочек одинакового цвета.

## Возможности

- Ввод с клавиатуры
- Генерация случайных данных
- Загрузка из файла
- Собственная реализация двусвязного списка
- Обработка исключений

## Сборка

```bash
g++ main.cpp BallGame.cpp DoublyLinkedList.cpp -o BallsGame
```

## Запуск

```bash
./BallsGame
```
